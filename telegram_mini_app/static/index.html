Оновлення даних

Запустити оновлення з ProZorro та OLX. Прогрес відображається нижче.

Увімкнути для цього запуску — отримання сторінок через Playwright замість HTTP. Спрацьовує для будь-якого оновлення з OLX (у т.ч. точкове).

Точкове оновлення (по джерелах, областях, типах)

Оберіть джерело, області та/або типи оголошень — оновлення виконається лише для обраного. Якщо області/типи не обрані — оновлюються всі.

Стан скрапера OLX

Оцінка того, наскільки повно парситься опис та параметри оголошень.

OLX скрапер (клікер) — експеримент

Запуск браузерного скрапера: імітація користувача (Playwright), збір даних у raw_olx_listings. Не змінює основний флоу оновлення.

Скрапер кадастру

Збір ділянок з kadastrova-karta.com. Center-first: спочатку Київ, потім решта України.

Після завершення завантаження кадастру: індекс місцезнаходження та кластеризація.

Аналітика цін

Перерахувати зведену аналітику (квартилі, середні ціни по містах). Виконується автоматично після оновлення даних.

Обробка аномальних цін

Виявлення оголошень з аномальними цінами (глобально, потім по місцевості). Результат — список для перевірки.

Планувальник

Планування автоматичного оновлення даних з ProZorro та OLX. Події виконуються ботом при його роботі. Зміни застосовуються після перезапуску бота.
